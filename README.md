Nesse repositório dei uma revisada de como o python trabalha com o básico de Orientação a Objetos. Estudei conceitos de: 
  - Classes
  - Objetos
  - Contrutor
  - Endereço na memória de referência do objeto criado
  - Atributos de classe
  - acesso de atributos
